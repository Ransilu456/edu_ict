The Mental Subnetting Trick
4 steps to instant answers
1
Find the Magic Octet

Divide CIDR by 8 → which octet has the boundary?
Class A (/8–/15) → 2nd octet | Class B (/16–/23) → 3rd | Class C (/24–/30) → 4th

2
Find Subnet & Host Bits

Subnet bits = CIDR − class boundary. Host bits = 8 − subnet bits.
E.g. /18 (Class B): subnet bits = 18 − 16 = 2, host bits = 8 − 2 = 6

3
Block Size (Magic Number)

Magic = 2host bits. For 6 host bits → 26 = 64.
Subnets increment by 64: .0, .64, .128, .192

4
Usable Hosts

Formula: 2(32 − CIDR) − 2. Subtract 2 for Network & Broadcast addresses.

Nyquist

Max Rate (Noiseless)

Maximum bit rate for bandwidth B and M signal levels:

Max bit rate = 2 × B × log₂(M)
Shannon

Max Rate (Noisy)

Theoretical maximum considering signal-to-noise ratio:

C = B × log₂(1 + SNR)
Manchester

Manchester Rule

IEEE 802.3 convention for guaranteed mid-bit clocking:

0 → Low→High (↑) 1 → High→Low (↓) Always mid-bit transition!
4B/5B

4B/5B Efficiency

4 data bits per 5-bit code, then NRZ-I on the wire:

Efficiency = 4/5 = 80% 100 Mbps wire rate → 80 Mbps actual data
