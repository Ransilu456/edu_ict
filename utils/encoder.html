ASK

Amplitude Shift Keying

Varies the amplitude (height) of the carrier. Bit 1 = full amplitude, Bit 0 = zero (or half). Simple but susceptible to noise.

1 → A·sin(2πft) 0 → 0 (or ½A·sin(2πft))
FSK

Frequency Shift Keying

Varies the frequency of the carrier. Bit 1 uses a higher frequency; Bit 0 uses the base frequency. More robust than ASK.

1 → A·sin(2πf₁t) [f₁ higher] 0 → A·sin(2πf₀t) [f₀ base]
PSK

Phase Shift Keying

Varies the phase of the carrier. Bit 1 is in phase (0°); Bit 0 is shifted by 180°. BPSK is the most common form.

1 → A·sin(2πft + 0°) 0 → A·sin(2πft + 180°)

NRZ-L

Non-Return-to-Zero Level

Level directly represents bit value: High = 1, Low = 0. Simple, but has DC component and long strings cause sync problems.

NRZ-I

NRZ Inverted

Transition at start of bit period for a 1; no transition for a 0. Self-synchronising for 1s. Still fails for long strings of 0s.

RZ

Return-to-Zero

Signal returns to zero in the middle of each bit period. Requires double bandwidth but provides self-clocking.

AMI

Alternate Mark Inversion

0 = zero voltage. 1 alternates between +V and −V. No DC component. Error detection built-in (violation detection).

4B/5B Encoding Table
Used in Fast Ethernet (100BASE-TX) — maps 4-bit nibbles to 5-bit codes

4B/5B ensures at most one leading zero and at most two trailing zeros, preventing long runs of zeros. The 5-bit codes are then passed through NRZ-I encoding on the wire.

Theory Reference — Signal Encoding
Key concepts for A/L ICT Paper 2

Why Encode Signals?

Data transmitted over a medium must be converted into signals the medium can carry. The choice of encoding scheme affects bandwidth efficiency, noise immunity, synchronisation, and error detection capability.

Key Terminology

TermDefinition
Baud RateNumber of signal changes per second (symbols/sec)
Bit RateNumber of data bits transmitted per second (bps)
Carrier WaveContinuous analog signal used as a base for modulation
BasebandDirect encoding of bits as voltage levels (NRZ, Manchester)
BroadbandEncoding bits onto a carrier (ASK, FSK, PSK)
DC ComponentNon-zero average voltage; problematic for transformers & capacitors
Self-ClockingSignal contains timing info (Manchester always has mid-bit transition)

Nyquist

Max Data Rate (Noiseless)

Maximum bit rate for a noiseless channel with bandwidth B and M signal levels:

Max bit rate = 2 × B × log₂(M)
Shannon

Max Data Rate (Noisy)

Shannon's theorem gives the theoretical maximum considering signal-to-noise ratio (SNR):

C = B × log₂(1 + SNR)
Manchester

Manchester Rule

A guaranteed mid-bit transition provides clock recovery. IEEE 802.3 convention:

Bit 0 → Low-to-High (↑) Bit 1 → High-to-Low (↓) Always a transition at mid-bit!
4B/5B

4B/5B Efficiency

Encodes 4 data bits into 5-bit codes to avoid long zero runs. Then uses NRZ-I.

Efficiency = 4/5 = 80% 100 Mbps wire rate → 80 Mbps actual data rate
