⚡ The Mental Subnetting Trick

1. Find the Magic Octet

Divide the CIDR prefix by 8. This tells you which octet contains the subnetting boundary (the "Magic Octet"):

  • Class A: /8 to /15 (2nd octet)
  • Class B: /16 to /23 (3rd octet)
  • Class C: /24 to /30 (4th octet)

2. Find Subnet & Host Bits

Subtract the base class boundary (8, 16, or 24) from the CIDR. E.g. For /18 (Class B), subnet bits = 18 - 16 = 2. Host bits in magic octet = 8 - 2 = 6.

3. Block Size (Magic Number)

Compute 2^(host bits in magic octet). For 6 host bits, 2^6 = 64. Subnets increment in this octet by 64: .0, .64, .128, .192.

4. Usable Hosts

Formula: 2^(total host bits) - 2. Total host bits is 32 - CIDR. Subtract 2 for the Network and Broadcast addresses.
